# Football Players Stats 2024/25

## Overview
Analysis of 2,694 players across 5 major European leagues 
using FBref data from Kaggle.

## Tools Used
- Python (Pandas, Plotly) — data cleaning and EDA
- Power BI — interactive dashboard

## Dataset
Source: Kaggle — hubertsidorowicz
165 columns · 2,854 rows · 5 leagues

## Project Structure
- Data cleaning — null handling, column renaming, 
  aggregation for multi-club players
- EDA — 6 phases, 36 charts
- Dashboard — 7 pages, interactive slicers

## Key Findings
- Mohamed Salah led all players with 29 goals and 18 assists
- Premier League had highest average tackles per player (36.85)
- Ligue 1 had highest average pass completion % (80.30%)
- Bundesliga players averaged the lowest pass completion (77.47%)
